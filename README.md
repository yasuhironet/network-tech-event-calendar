# ネットワーク技術関連イベントカレンダー

ネットワーク技術関連のイベントカレンダー。

## 方針

- 著名なネットワーク関連イベント、トップカンファレンスを網羅し、
  技術者にリマインダするためのものです。
- 基本的にイベントの追加・削除の依頼は受け付けません。
  個人的なセレクションです。
